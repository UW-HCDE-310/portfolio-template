Japanese Dictionary and Flashcards

June 14th, 2019 Naomi Johnson

Project

Learning a new language involves looking up new words in a dictionary--- but sometimes students find themselves looking up the same word again and again if they do not learn it. This is particularly difficult for students learning Japanese. In addition to understanding the english meaning of a given word, they will need to learn the pronunciation of the word (written in either hiragana or katakana) as well as the written kanji (symbols that represent the meaning of the word, not the pronunciation).

Acomplishments

My team and I created a website that is not only a Japanese dictionary (complete with xyz thousand entries from the EDICT files from the Electronic Dictionary Research and Development Group at Monash University), but also lets users login to create flashcard decks, practice the flashcards, and record whether they knew the answer or not. You can experience our prototype with AdobeXd, or view our code. Unfortunately, our AWS machine has expired and needs to be set up again.

For this project, I wrote python that converted the JSON files into MySQL statements ("meta programing") to fill our databases. I learned not everyone uses UTF-8, and I spent a long time working to get all the different coding languages to use Japanese successfully.

I learned how to use AdobeXd to create a fucntioning protoype. I ran qualitative user studies.

Results

I earned an "A" on this project and in this course, as did my collaborators.
